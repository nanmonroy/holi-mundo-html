GitHub es una plataforma donde puedes guardar tus proyectos, colaborar con otros programadores y tener todo tu código seguro en la nube.

💻 Visual Studio Code

Visual Studio Code es un editor de código superligero, personalizable y amigable, ideal para escribir tus programas y conectarte fácilmente con GitHub desde ahí.

🎮 Discord

Discord no solo es para gamers, también sirve para crear comunidades de programadores, organizar proyectos en grupo y compartir avances en canales especiales.
