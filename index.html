🐱 GitHub

Biografía: GitHub nació en 2008 como una plataforma para alojar código con control de versiones usando Git.

Funcionalidad: Guardar tus proyectos, colaborar con otros y llevar registro de cambios.

Beneficio: Comunidad activa, historial de versiones y colaboración en tiempo real.

💻 Visual Studio Code

Biografía: VS Code fue creado por Microsoft en 2015 como un editor de código gratuito.

Funcionalidad: Soporte para múltiples lenguajes, terminal integrada, extensiones.

Beneficio: Personalizable, ligero, poderoso y fácil de usar.

🎮 Discord

Biografía: Discord nació en 2015 como una app para gamers, pero ahora es usada por comunidades de todo tipo.

Funcionalidad: Chats de voz, texto, videollamadas, bots y servidores.

Beneficio: Espacio seguro para aprender, colaborar y compartir.
